> CLI app to scrap and download open data from São José dos Campos

## Installation

```sh
$ npm install -g sjcdigital
```

## Usage

Open your terminal and run: 
```sh
sjcdigital bus
```
and it will download all the bus data and put it inside a folder called `onibus-data`
